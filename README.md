Repository template for **Assignment 3** of the Programming Languages course (*Programmiersprachen*).
This assignment covers pointers, generic programming with templates, containers, and the Standard Template Library (STL).
Students implement a doubly linked list (`List`) as a template class, including `ListNode`, `ListIterator`, and `List` with pre-written [doctest](https://github.com/doctest/doctest) unit tests for the core operations (`push_back`, `pop_back`, `push_front`, `pop_front`, `front`, `back`, `begin`, `end`, and iterator operations).
